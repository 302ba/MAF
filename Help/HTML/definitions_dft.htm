DFT

 Top  Previous  Next

The DFT (Dynamic Function Table) contains all code links in a MAF based application. There are 2 versions of the DFT : an un-compiled version in the data storage from which the second version is being compiled from when the application starts and a user logs on. The compilation includes only functions the current user has access to, preventing the user centralized from calling functions he shouldn't have access to.
