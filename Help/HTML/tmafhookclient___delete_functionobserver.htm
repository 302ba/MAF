__Delete_FunctionObserver

 Top  Previous  Next

Deletes an previously installed observer for a DynamicFunction.

 

Class

TmafHookClient

 

Syntax

 

function __Delete_FunctionObserver(nHookID, nSubHookID: Integer; AReceiver: TComponent = nil): Integer;

 

Remarks

nHookID defines the ID of the DynamicFunction to observe.

nSubHookID defines a SubHook to observe. Is this parameter is 0, the whole DynamicFunction will be observed.

AReceiver can contain a receiver component for the message sent by the TmafHookManager. For application development the parameter can just be left blank, but components should use this to intercept these messages instead of using the event fired.

The result is an error code. If the execution was successful, ERR_NO_ERROR is returned.

 

The parameter should be given here exactly as they were given when installing the function observer. If the parameter do not fit, the observer will not be de-installed and might call into an already destroyed callback.
