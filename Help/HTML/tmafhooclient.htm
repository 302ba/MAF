TmafHookClient

The TmafHookClient ist actually used to connect to several Manager-Components :

 

TmafHookManager

TmafDataStorage

TmafGlobalVars

 

 

 

 

Properties

 

Name

Description

HookClientOptions

hcoProcessOnIdle to receive OnIdle events from the MainForm

hcoClearParamStack to clear the param stack after every call

Parameter

connected TmafParameters Component

Streamer

TTemplateStreamer to have easy access to streamed data

 

Methods

 

Name

Description

ExecuteHook

Executes a Hook with all its SubHooks

ExecuteUniqueID

Executes a specific SubHook through its UniqueID

GetGlobalString

Returns a previously saved String from the manager component TmafGlobalVars

SetGlobalString

Saves a String in the manager component TmafGlobalVars

GetGlobalInteger

Returns a previously saved Integer from the manager component TmafGlobalVars

SetGlobalInteger

Saves a Integer in the manager component TmafGlobalVars

GetGlobalObject

Returns a previously saved Object from the manager component TmafGlobalVars

SetGlobalObject

Saves an Object in the manager component TmafGlobalVars

GetGlobalBoolean

Returns a previously saved Boolean from the manager component TmafGlobalVars

SetGlobalBoolean

Saves a Boolean in the manager component TmafGlobalVars

LoadTemplate

Loads a template stored through the manager component TmafDataStorage

SaveTemplate

Saves a template in the manager component TmafDataStorage

PushEvents

Saves the current events into a stack and clears the events

PopEvents

Restores previously stored events from the stack

PushVar

Saves a variable to a stack

PopVar

Loads a variable from a stack and deletes it there

DeleteTemplate

Deletes a template

RegisterCallback

Registers a callback notification for a specific global variable

UnregisterCallback

Deregisters a previously registered callback notification for a specific global variable

__Query_Manager

Executes a query to the TmafHookManager component

__Add_FunctionObserver

Registers a function observer to be notified, when something in the DFT changes (add or delete).

__Delete_FunctionObserver

Deregisters a function observer.

 

Events

 

Name

Description

AfterCallRouter

Fired when ExecHook or ExecUniqueID is called after the call returns from the TmafHookManager

AfterExecHook

Fired after every SubHook within a Dynamic Function (Hook)

AfterQuery

Fired after execution of __Query_Manager

BeforeCallRouter

Fired before a Hook or UniqueID is executed

BeforeExecHook

Fired before the execution of every SubHook within a Hook

BeforeQuery

Fired before the execution of __Query_Manager

OnDFTChanged

The DFT has changed (after install/uninstall of a module)

OnFunctionObserver

FunctionObserver fires this event, if at least one Hook or SubHook is on our watch list (added through __Add_FunctionObserver)

OnGlobalVarChanged

A global variable in TmafGlobalVars has changed

OnGlobalVarDeleted

A global variable in TmafGlobalVars has been deleted

OnIdle

OnIdle in the MainForm occurs and the HookClient is notified because in HookClientOptions the option hcoProcessOnIdle is set

OnStreamReadAttribute

OnStreamVersion

OnStreamVersionError

OnStreamWriteAttribute

OnManagerContainerError
