TmafManagerLoader class overview

 

Properties

 

Name

Description

AutoLoad

Causes the component to load automatically the manager and connects to them

DataLoadMethod

Defines from where the component loads the information about the manager to be loaded

INISettings

Settings for the INI file if it is chosen in DataLoadMethod

Manager

List of manager names to load

ManagerLoaderOptions

Defines the loader options

ManagerSubDirectory

Defines the manager sub directory

RegistrySettings

Settings for the registry if it is chosen in DataLoadMethod

 

Methods

 

Name

Description

AddManager

Adds a manager.

DeleteManager

Deletes a manager.

ConnectManager

Connects to all manager registered.

DisconnectManager

Disconnects from all manager loaded.

 

Events

 

Name

Description

AfterConnect

Fired after connecting to a manager.

BeforeConnect

Fired before connecting to a manager.

BeforeLoadingManager

Fired before starting to load all registered manager.

OnConnect

Fired when connected to a manager successfully.

OnDataLoad

Fired for every registered manager when DataLoadMethod is set to dlmUserdefined.

OnDisconnect

Fired when disconnecting from a manager.

OnLoadingFinished

Fired when all registered manager are loaded.

OnLoadManagerError

Fired when the loading of a manager fails.
