BeforeFormCreate

 Top  Previous  Next

Fired before a form is created by the TmafModuleController. Obviously it will be only fired, if the TmafModuleController has a form associated with the current SubHook and the default action for the SubHook is one of the Form-Creation-Methods.

 

Class

TmafModuleController

 

Syntax

 

property OnBeforeFormCreate : TBeforeFormNotifyEvent;

 

TBeforeFormNotifyEvent = procedure(Sender: TObject; QHS: pQHS; nSubHookID: Integer; pData: Pointer; var CanContinue: Boolean) Of Object;

 

Remarks

 

With the variable QHS you receive the pointer to a TQueryHandlerStruct record from the caller and can access all variables and parameters within it to check, if the form creation can continue.

pData is a data pointer to the structure assigned to the SubHook within the local ModuleController.

CanContinue is default true, to inform the ModuleController to not create the form for some reason, just set it to false before leaving the event.
