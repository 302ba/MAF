DataLoadMethod

 Top  Previous  Next

Defines the method from where the information about the manager to load will be taken from.

 

Class

TmafManagerLoader

 

Syntax

 

property DataLoadMethod: TDataLoadMethod default [dlmUserdefined];

 

Remarks

The component defines three sources to load the manager information from :

 

1.INI file

2.Registry

3.User defined

 

While the first two can be set up that no further interaction during the loading process is necessary, the third will fire an OnDataLoad event for each manager to load, where the file name of the manager to be loaded is expected.
