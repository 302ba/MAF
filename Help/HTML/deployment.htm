Deployment

MAF Applications and all modules always have to be built with runtime packages of at least RTL, VCL, MAF_Base and MAF_Visual. These BPL files have to be deployed with your application.

 

Remember, that all files use lib - suffixes, so every BPL file contains the Delphi-Version number :

 

Delphi 2010 : VCL140.bpl, RTL140.bpl, MAF_Base140.bpl, MAF_Visual140.bpl

Delphi 2009 : VCL120.bpl, RTL120.bpl, MAF_Base120.bpl, MAF_Visual120.bpl

Delphi 2007 : VCL100.bpl, RTL100.bpl, MAF_Base100.bpl, MAF_Visual100.bpl

Delphi 2006 : VCL100.bpl, RTL100.bpl, MAF_Base100.bpl, MAF_Visual100.bpl

Delphi 2005 : VCL90.bpl, RTL90.bpl, MAF_Base90.bpl, MAF_Visual90.bpl

Delphi 7      : VCL70.bpl, RTL70.bpl, MAF_Base70.bpl, MAF_Visual70.bpl
